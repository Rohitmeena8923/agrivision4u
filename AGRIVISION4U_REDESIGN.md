## ✨ Design System Highlights

| Feature | Before | After |
|---------|--------|-------|
| **Theme** | Plain light/white | Dark premium (#030712) |
| **Colors** | Basic green/brown | Emerald (#10B981) + Cyan (#06B6D4) + Amber (#F59E0B) |
| **Typography** | Default fonts | Outfit (display) + Inter (body) + Space Grotesk (data) |
| **Cards** | Flat basic cards | Glassmorphic cards with backdrop blur + glow borders |
| **Backgrounds** | Solid white | Mesh gradients + grid patterns + aurora glow orbs |
| **Buttons** | Plain colored | Gradient CTAs with hover color inversion |
| **Text Effects** | Plain text | Multi-color gradient text headings |
| **Animations** | Minimal | Framer Motion staggered reveals, floating, pulsing glows |
| **Navbar** | Basic sticky | Glassmorphic with animated dropdown + gradient CTA |
| **Footer** | Plain text | Dark premium with glassmorphic social icons |

## 🎯 Key Design Elements Used

1. **Glassmorphism** — Semi-transparent cards with `backdrop-filter: blur(20px)`
2. **Mesh Gradients** — Multi-point radial gradients for vibrant backgrounds
3. **Aurora Glow Orbs** — Animated floating color spheres
4. **Grid Pattern Overlay** — Subtle technical grid texture
5. **Gradient Text** — Emerald → Cyan → Amber gradient for headings
6. **Glow Borders** — Green-tinted borders with box-shadow glow on hover
7. **Staggered Animations** — Sequential reveal of cards/elements
8. **Custom Scrollbar** — Dark themed with emerald hover accent
9. **Noise Texture** — Subtle SVG noise overlay for depth
10. **Space Grotesk** — Data/stats font for a technical, premium feel
